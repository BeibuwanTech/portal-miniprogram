# 北部湾城市群综合科技服务平台门户小程序

## 首页功能

### 首页第一屏

![HomePage First](/doc/images/home1.jpg)

- 地区选择：可以切换到北部湾城市群的特定城市，查看该城市相关的信息流
- 搜索功能：基于知识图谱的搜索引擎
- 主要轮播区：科技服务业最新资讯，平台个性化推荐
- 公告区：平台重要公告通知
- 功能菜单区：研究开发、知识产权等五大资源池集成应用示范入口

### 首页第二屏

![HomePage Second](/doc/images/home2.jpg)

- 特色功能首页展示区（国家强制性标准数据库）：基于检验检测认证资源池构建的国家标准快速查询服务
- 今日推荐：平台广告区域

### 首页第三屏

![HomePage Third](/doc/images/home3.jpg)

- 直达入口：科技服务入驻服务，科技服务需求发布功能快捷入口
- 科技服务：基于研究开发、知识产权等五大科技服务资源池的科技服务集成，"为你推荐"栏目采用个性化推荐方式推送科技服务资源信息流
